## 二、核心业务表

| 表名 | 说明 | 关键关联 |
|---|---|---|
| `students` | 学生基础资料 | 被 `class_students.student_id`、`attendance.student_id` 关联 |
| `teachers` | 教师基础资料 | 被 `classes.teacher_id`、`lessons.teacher_id`、`users.teacher_id` 关联 |
| `lesson_types` | 课程类型和课时规则 | 被 `lessons.lesson_type_id` 关联 |
| `course_presets` | 预设课程库 | 被 `lessons.course_preset_id` 关联，包含课程大类、阶段、节次、课程名称 |
| `classes` | 班级基础资料 | 被 `class_students.class_id`、`lessons.class_id` 关联 |
| `class_students` | 班级学生关联表 | 连接 `classes` 与 `students` |
| `lessons` | 每节实际课次 | 关联班级、教师、课程类型、预设课程 |
| `lesson_details` | 每节课详细课程记录 | 通过 `lesson_id` 一对一关联 `lessons` |
| `attendance` | 学生每节课签到记录 | 连接 `lessons` 与 `students` |
| `users` | 登录账号和角色 | 可关联 `teachers` |
| `operation_logs` | 操作日志 | 可关联 `users` |

## 四、常用联表查询

常用 SQL 已整理到：

```text
E:\moran_project\class_worker\sql\common_queries.sql
```

示例：查询学生课时余额。

```sql
SELECT
    student_id,
    student_name,
    class_names,
    total_hours,
    consumed_hours,
    remaining_hours,
    last_lesson_date
FROM v_students_summary
WHERE status = 'active'
ORDER BY remaining_hours ASC, student_name;
```

示例：查询指定日期课次详情。

```sql
SELECT
    lesson_date,
    start_time,
    end_time,
    class_name,
    teacher_name,
    lesson_type_name,
    expected_count,
    arrived_count,
    leave_count,
    absent_count,
    attendance_rate
FROM v_lessons_detail
WHERE lesson_date = '2026-05-16'
ORDER BY start_time, class_name;
```

示例：查询签到明细。

```sql
SELECT
    lesson_date,
    start_time,
    class_name,
    teacher_name,
    student_name,
    attendance_status,
    deduct_hours,
    attendance_remark
FROM v_attendance_detail
WHERE lesson_date BETWEEN '2026-05-01' AND '2026-05-31'
ORDER BY lesson_date, start_time, class_name, student_name;
```

## 六、后续数据库优化建议

1. 增加课时购买流水表，例如 `student_hour_transactions`。
2. 增加试听跟进表，例如 `trial_students`、`trial_followups`。
3. 增加学生维度课堂反馈表，例如 `student_feedback`，用于补充单个学生的个性化表现。
4. 增加自动备份任务表或系统设置表，例如 `system_settings`。
5. 如果后续并发人数增多，可从 SQLite 平滑迁移到 MySQL 或 PostgreSQL。

## 七、与 Vue 前端和 API 的关系

当前系统已经新增 Vue3 + Element Plus 前端，前端通过 Flask `/api` 接口访问数据库。

| API | 主要数据表或视图 | 说明 |

| `/api/students` | `students`、`class_students`、`attendance` | 学生列表、课时余额 |

| `/api/classes/<id>/students` | `class_students`、`students` | 班级学生维护 |
| `/api/classes/<id>/lessons/generate` | `classes`、`lessons`、`lesson_types`、`course_presets` | 按日期范围和星期批量生成课次 |
| `/api/lesson-types` | `lesson_types` | 课程类型维护 |
| `/api/course-presets` | `course_presets`、`lesson_types` | 预设课程维护 |

| `/api/course-presets/import-template` | 生成 Excel 模板 | 下载预设课程导入模板 |
| `/api/lessons` | `lessons`、`classes`、`teachers`、`lesson_types`、`course_presets`、`attendance` | 课次列表和新增课次 |
| `/api/lessons/<id>/cancel` | `lessons` | 取消课次 |

| `/api/statistics` | `attendance`、`lessons`、`classes`、`teachers`、`lesson_types`、`students` | 统计汇总和明细 |

## 八、当前数据库注意事项

1. SQLite 不适合多人高并发写入，但适合当前本地或局域网小规模使用。
2. 删除类操作多数采用停用、归档或移出，不建议直接物理删除历史业务数据。
3. 使用数据库软件直接修改数据前，必须先备份 `attendance.db`。
4. 如果通过数据库软件执行写入语句，请先执行：

```sql
PRAGMA foreign_keys = ON;
```

5. 如果将来迁移 MySQL/PostgreSQL，建议优先迁移表结构和视图语义，再改 Flask 数据访问层。
