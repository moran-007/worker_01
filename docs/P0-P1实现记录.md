## 5. 新增/变更接口

| 接口 | 说明 |
| --- | --- |
| `GET /api/permissions` | 返回数据库权限目录和角色授权 |
| `POST /api/permissions` | 超级管理员新增或修改权限点 |
| `PUT /api/permissions/roles/<role_code>` | 超级管理员修改角色权限 |
| `GET /permissions` | 超级管理员查看权限矩阵、手动勾选身份权限、自定义新增身份 |
| `GET /student` | 学生端入口，学生登录后默认跳转 |
| `GET /api/student/me` | 学生查看自己的课程、班级和学习记录；无 `student_portal.hours.view` 时课时字段脱敏 |
| `GET /students/<student_id>/portal` | 教师/管理员预览学生端页面，教师只能预览本人班级学生 |
| `GET /api/students/<student_id>/portal` | 教师/管理员获取指定学生端数据，教师范围受班级归属限制 |
| `GET /parent` | 家长端入口，家长登录后默认跳转 |
| `GET /api/parent/me` | 家长查看账号关联学生的课时、班级、排课和学习记录 |
| `GET /students/<student_id>/parent` | 教师/管理员预览家长端页面，教师只能预览本人班级学生 |
| `GET /api/students/<student_id>/parent` | 教师/管理员获取指定家长端数据，教师范围受班级归属限制 |
| `GET /api/integrations/accounts` | 查看外部账号绑定 |
| `POST /api/integrations/accounts` | 绑定 Hydro 等外部账号 |

## 7. 验证结果

已通过：

- Python 语法编译。
- 应用初始化数据库。
- `users.student_id` 自动迁移。
- 权限表自动种子初始化。
- 超级管理员拥有全部动态权限。
- 管理员 API 登录成功。
- `/api/me` 返回动态权限。
- `/api/permissions` 返回 8 个默认身份，并包含 `parent_portal.view` 与 `student_portal.hours.view`。
- 学生账号 `student01 / student123` 可登录，绑定 `student_id=13`，登录后自动跳转 `/student`。
- `/api/student/me` 只返回当前登录学生自己的班级、近期课程和学习记录，不返回同班其他学生信息；默认学生身份无 `student_portal.hours.view`，课时字段返回 `null`。
- 学生账号访问 `/api/permissions`、`/api/dashboard`、`/students`、`/api/students/14/portal` 均返回 403。
- Vue 管理端登录页已识别学生角色，学生账号从 `http://127.0.0.1:5173/login` 登录后会跳转到后端学生端 `http://127.0.0.1:8000/student`。
- 家长账号 `parent01 / parent123` 可登录，绑定 `student_id=13`，登录后自动跳转 `/parent`。
- `/api/parent/me` 只返回家长账号关联学生的课时、班级、近期排课和学习记录；访问其他学生家长端数据返回 403。
- Vue 管理端登录页已识别家长角色，家长账号从 `http://127.0.0.1:5173/login` 登录后会跳转到后端家长端 `http://127.0.0.1:8000/parent`。
- 教师账号 `teacher01 / teacher123` 可预览本人班级学生 `student_id=13` 的学生端，访问其他教师班级学生 `student_id=14` 返回 403。
- 教师账号可预览本人班级学生的家长端，访问其他教师班级学生的家长端返回 403。
- 管理员账号 `admin / admin123` 可预览任意学生端。
- 后端 `/login` 对管理类账号默认跳转 Vue 管理端 `http://127.0.0.1:5173/`，不再落到旧 Flask 管理页；明确访问 `/permissions` 等备用页时保留登录后回跳。
- 超级管理员可新增/修改权限点。
