# 功能模块路径审查清单

生成时间：2026-06-06

## 1. 本地启动与主入口

| 模块 | 入口/路径 | 说明 |
| --- | --- | --- |
| 一键启动 | `restart_all_services.bat` / `restart_all_services.ps1` | 关闭 8000、5173、3000、5174、8601 后启动主系统、Vue、OJ_text、Scratch GUI。 |
| 主系统后端 | `app/app.py` | Flask API、Vue shell 托管、Scratch 编辑器桥接页。 |
| Vue 管理端 | `app/frontend/src/main.js` | Vue3 + Element Plus 管理端入口。 |
| 路由 | `app/frontend/src/router/index.js` | 登录、管理端、教师备课、上课入口、学生端、学生课程二级页、家长端。 |
| 全局布局 | `app/frontend/src/components/AppShell.vue` | 左侧菜单、顶部标题、退出登录、修改密码。 |
| 全局样式 | `app/frontend/src/styles/app.css` | EduPro 风格、工作台、学生端、备课/上课入口样式。 |

## 8. 后端数据范围与安全点

| 模块 | 路径 | 说明 |
| --- | --- | --- |
| 教师范围过滤 | `app/app.py::apply_teacher_scope` | 教师只能拉取自己授课范围内的数据。 |
| 课次访问校验 | `app/app.py::ensure_lesson_access` | 教师访问课次详情、签到、作品列表前校验课次归属。 |
| 学生范围校验 | `app/app.py::ensure_student_access` | 学生/家长只能查看关联学生；教师只能查看自己班级学生。 |
| Scratch 作品校验 | `app/app.py::ensure_scratch_work_access` | 学生本人可保存/提交；家长只读；教师只读并可点评；已点评作品学生不可再改。 |
| Scratch 模板校验 | `app/app.py::ensure_scratch_template_preview_access` | 学生/家长只能预览自己关联课次绑定的模板。 |
| 操作日志 | `app/app.py::log_operation` | 登录、上传、绑定、保存、提交、点评都会记录到 `operation_logs`。 |
